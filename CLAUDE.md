## Text colour rule

- On dark, black, or image backgrounds: body and heading text must be plain white (`text-white`). NEVER use `text-primary` for text on these backgrounds.
- `text-primary` is only allowed for text on white/light backgrounds (e.g. a `bg-white` section).
